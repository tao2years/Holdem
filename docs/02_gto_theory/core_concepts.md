# GTO 核心概念

本章以“总-分-总”结构总结 GTO 的核心概念，并给出学习路径。

## 本节目的

- 解释 GTO 的含义与目标
- 说明均衡与策略平衡

## 关键概念

- 纳什均衡：双方最优应对下的稳定策略
- 混合策略：以概率选择行动
- 无可剥削性：对手无法在长期获利地偏离

## 均衡直觉

- 均衡不是“每手都赢”，而是“长期不被剥削”
- 对手偏离，你不必调整也不损失 EV

## 策略表达

- 以范围为单位表达
- 以频率为单位执行
- 以 EV 作为检验

## 训练建议

- 先构建稳定范围，再谈动作与尺度
- 保持频率稳定，避免过度随意

## 学习建议

- 先理解博弈论基础与均衡概念
- 再结合实际场景理解范围与频率

## 总结

GTO 是一种“难以被剥削”的策略框架，理解均衡与混合策略是关键。
