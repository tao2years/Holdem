# GTO 理论总览

本章以"总-分-总"方式整理 GTO 理论入口，提供完整的学习路径与核心概念索引。

## 本节目的（总）

- 建立 GTO 理论框架与学习路径
- 指出核心概念与关键文档
- 提供系统化的学习顺序

## 核心概念（分）

### 1. GTO 核心概念

**文档**：`core_concepts.md`

**内容**：
- GTO 的本质与目标
- 纳什均衡详解
- 混合策略原理
- 不可剥削性验证

- 理解混合策略的必要性与实现方法

### 2. 范围与频率

**学习重点**：
- 掌握各位置的翻牌前范围
- 理解范围在多街中的演化
- 掌握频率分配的原则与方法

### 3. 42 原则

**文档**：`rule_42.md`

**学习重点**：
- 建立系统化的训练计划

- 持续改进策略

   - 通过复盘持续改进

## 学习建议（分）

### 理论学习

**重点**：
- 深入理解核心概念，不要死记硬背
- 理解概念之间的联系，建立知识体系
- 结合实际场景理解理论

**方法**：
- 阅读文档，理解核心概念
- 完成练习题，验证理解
- 使用 Solver 验证理论

### 实践应用

**重点**：
- 在实战中应用 GTO 理论
- 使用 Solver 验证关键决策
- 通过复盘持续改进

**方法**：
- 每周应用 2-3 次 GTO 理论
- 每周使用 Solver 验证 1 次
- 每周复盘 5-10 个案例

### 持续改进

**重点**：
- 记录偏离 GTO 的决策
- 分析偏离原因
- 制定改进方案

**方法**：
- 建立错误记录表
- 定期分析错误原因
- 调整训练计划
