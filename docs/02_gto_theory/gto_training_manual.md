# GTO 训练手册（总览）

本手册从零构建不可被剥削的德州扑克体系，聚焦 GTO 理论框架内的最优策略与训练方法，覆盖翻牌前到河牌的完整策略体系。

## 本节目的

- 统一“不可被剥削”的训练目标
- 提供分阶段训练路径与复盘方法
- 形成从理论到实战的闭环

## 训练总框架

### 1. 认知层

- 概念：均衡、范围、频率、EV、赔率、后悔值
- 原则：先范围、后行动；先频率、后尺度

### 2. 技能层

- 翻牌前：按位置建立稳定开池与防守范围
- 翻牌后：根据范围与牌面构建下注/过牌频率
- 河牌：价值与诈唬的边界管理

### 3. 复盘层

- 以“范围—频率—尺度—EV”四步回溯决策
- 记录偏离均衡的原因与修正方式

## 训练方法

- 分街训练：每次只训练一条街的策略模板
- 位置训练：按 UTG/MP/CO/BTN/SB/BB 分别训练
- 案例训练：用具体牌例验证决策流程

## 训练节奏建议

- 每周固定两次“翻牌前范围”训练
- 每周固定一次“翻后频率与尺度”训练
- 每周至少一次复盘与整理

## 复盘模板

1. 范围：我与对手的范围是否合理
2. 频率：我是否过度下注或过度过牌
3. 尺度：下注尺度是否与范围匹配
4. EV：此选择是否为正期望

## 总结

GTO 训练不是记忆牌谱，而是建立稳定的决策流程：先确定范围，再分配频率，最后选择尺度，并用 EV 验证。
