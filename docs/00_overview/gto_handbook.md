# GTO 训练手册总览

本手册从零构建不可被剥削的德州扑克 GTO 体系，聚焦博弈论最优策略，不讨论针对特定对手的剥削性调整。内容覆盖翻牌前到河牌的完整策略框架，并提供可操作的训练方法与实战案例。

## 本节目的

- 建立统一的 GTO 训练框架
- 提供可执行的训练路径与复盘方法
- 形成从理论到实战的闭环

## 结构导航

- `docs/01_basics/`：概率、EV、底池计算等基础工具
- `docs/02_gto_theory/`：均衡、范围与频率等核心概念
- `docs/03_advanced/`：翻牌前范围、翻后策略、下注尺度、局势判断
- `docs/03_advanced/case_studies.md`：实战案例与训练任务

## 使用方式

1. 先完成基础工具与核心概念的学习
2. 按位置与街道训练策略模板
3. 使用案例进行复盘与验证

## 总结

本手册的目标是形成“可学、可练、可验证”的 GTO 训练体系，最终建立稳定的不可剥削策略框架。
