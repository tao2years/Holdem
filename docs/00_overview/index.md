# 总览

本仓库用于系统记录德州扑克 GTO 理论与 Solver 实现，内容采取“总-分-总”的结构，便于学习、复盘与工程实现。

## 本节目的

- 说明仓库定位与边界
- 提供内容入口与阅读顺序

## 内容入口

- `docs/01_basics/`：规则、术语、基础概率
- `docs/02_gto_theory/`：GTO 核心概念与推导
- `docs/03_advanced/`：范围构建、下注尺度与对抗
- `docs/04_solver/`：Solver 架构与算法实现
- `docs/05_experiments/`：实验与验证
- `docs/99_notes/`：碎片化笔记
- `docs/00_overview/gto_handbook.md`：GTO 训练手册入口

## 结论

本仓库以知识体系与工程实现双线推进，先完成基础知识与最小可运行 Solver，再逐步扩展到多街与抽象策略。
