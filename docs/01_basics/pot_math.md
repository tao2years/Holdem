# 底池计算与常用比例

本章以"总-分-总"结构详细整理底池计算与常用比例，用于支撑 GTO 决策中的 EV 判断。

## 本节目的（总）

- 提供清晰的底池计算方法与公式
- 建立常用比例与快速估算能力
- 掌握底池计算在 GTO 决策中的应用

## 底池基础（分）

### 底池大小的定义

**底池（Pot）**：当前对局中所有玩家已投入筹码的总和。

**底池组成**：
- **盲注**：小盲和大盲的初始投入
- **下注**：各玩家的下注金额
- **跟注**：各玩家的跟注金额
- **加注**：各玩家的加注金额

**底池计算**：
```
底池 = 盲注 + 所有下注 + 所有跟注 + 所有加注
```

### 跟注成本

**跟注成本（Call Cost）**：当前行动者为继续对局需投入的筹码数量。

**跟注成本计算**：
```
跟注成本 = 对手下注金额 - 你已经投入的金额
```

**例子**：
- **底池**：100
- **对手下注**：50
- **你已经投入**：0
- **跟注成本**：50

### 下注尺度

**下注尺度（Bet Size）**：下注金额相对于底池的比例。

**常用尺度**：
- **1/3 Pot（33%）**：下注为底池的 1/3
- **1/2 Pot（50%）**：下注为底池的 1/2
- **2/3 Pot（67%）**：下注为底池的 2/3
- **1 Pot（100%）**：下注等于底池
- **全下（All-in）**：投入所有筹码

**下注尺度计算**：
```
下注尺度 = 下注金额 / 底池大小
```

## 常用比例（分）

### 1/3 Pot（33%）

**计算**：
- **底池 100**：下注 33
- **底池 200**：下注 67
- **底池 300**：下注 100

**使用场景**：
- **范围优势不明显**：使用小尺度保护范围
- **高频下注**：小尺度配合高频下注
- **干燥牌面**：范围优势明显，使用小尺度

**特点**：
- **给对手合适赔率**：对手容易跟注
- **保护范围**：避免暴露手牌强度
- **控制底池**：保持底池可控

### 1/2 Pot（50%）

**计算**：
- **底池 100**：下注 50
- **底池 200**：下注 100
- **底池 300**：下注 150

**使用场景**：
- **范围优势明显**：使用中等尺度获取价值
- **中频下注**：中等尺度配合中频下注
- **中等牌面**：范围优势适中，使用中等尺度

**特点**：
- **平衡价值与诈唬**：给对手不合适但可接受的赔率
- **获取价值**：从对手跟注中获取价值
- **控制底池**：保持底池可控

### 2/3 Pot（67%）

**计算**：
- **底池 100**：下注 67
- **底池 200**：下注 133
- **底池 300**：下注 200

**使用场景**：
- **范围极化**：强牌与弱牌比例增加
- **低频下注**：大尺度配合低频下注
- **湿润牌面**：给听牌不合适赔率

**特点**：
- **给对手不合适赔率**：迫使对手弃牌或跟注错误
- **最大化价值**：从对手跟注中最大化价值
- **控制底池**：保持底池可控

### 1 Pot（100%）

**计算**：
- **底池 100**：下注 100
- **底池 200**：下注 200
- **底池 300**：下注 300

**使用场景**：
- **范围极度极化**：只有强牌和弱牌
- **极低频下注**：大尺度配合极低频下注
- **河牌价值下注**：最大化价值获取

**特点**：
- **给对手极不合适赔率**：迫使对手弃牌
- **最大化价值**：从对手跟注中最大化价值
- **控制底池**：保持底池可控

## 快速估算（分）

### 底池赔率计算

**底池赔率（Pot Odds）**：当前底池与跟注成本的比值。

**计算公式**：
```
底池赔率 = 底池大小 / 跟注成本
```

**例子**：
- **底池**：100
- **对手下注**：50
- **跟注成本**：50
- **底池赔率**：100 / 50 = 2:1

### 所需胜率计算

**所需胜率（Required Equity）**：跟注所需的最低胜率。

**计算公式**：
```
所需胜率 = 跟注成本 / (底池 + 对手下注 + 跟注成本)
```

**简化公式**：
```
所需胜率 = 跟注成本 / (底池 + 跟注成本)
```

**例子**：
- **底池**：100
- **对手下注**：50
- **跟注成本**：50
- **所需胜率**：50 / (100 + 50 + 50) = 50 / 200 = 25%

### 快速估算方法

**方法 1：心算常见比例**

**1/3 Pot 下注**：
- **所需胜率**：约 25%
- **记忆**：1/3 Pot ≈ 25% 胜率

**1/2 Pot 下注**：
- **所需胜率**：约 33%
- **记忆**：1/2 Pot ≈ 33% 胜率

**2/3 Pot 下注**：
- **所需胜率**：约 40%
- **记忆**：2/3 Pot ≈ 40% 胜率

**1 Pot 下注**：
- **所需胜率**：约 50%
- **记忆**：1 Pot ≈ 50% 胜率

**方法 2：快速计算**

**步骤**：
1. **估算底池大小**：快速估算当前底池
2. **计算下注比例**：下注金额 / 底池大小
3. **估算所需胜率**：根据比例估算所需胜率
4. **比较实际胜率**：估算实际胜率，比较判断

**例子**：
- **底池**：约 100
- **对手下注**：约 50
- **下注比例**：50 / 100 = 50%
- **所需胜率**：约 33%
- **实际胜率**：估算约 40%
- **判断**：40% > 33%，应该跟注

## 常见底池计算（分）

### 翻牌前底池

**初始底池**：
- **小盲 + 大盲**：1 + 2 = 3（1/2 盲注结构）
- **例子**：1/2 盲注，初始底池 = 3

**开池后底池**：
- **初始底池 + 开池金额**：3 + 2.5 = 5.5
- **例子**：1/2 盲注，开池 2.5BB，底池 = 5.5BB

**3-Bet 后底池**：
- **开池后底池 + 3-Bet 金额**：5.5 + 8 = 13.5
- **例子**：开池 2.5BB，3-Bet 到 8BB，底池 = 13.5BB

### 翻后底池

**翻牌后底池**：
- **翻牌前底池 + 翻牌下注 + 跟注**：5.5 + 2 + 2 = 9.5
- **例子**：翻牌前底池 5.5BB，下注 2BB，跟注 2BB，底池 = 9.5BB

**转牌后底池**：
- **翻牌后底池 + 转牌下注 + 跟注**：9.5 + 5 + 5 = 19.5
- **例子**：翻牌后底池 9.5BB，下注 5BB，跟注 5BB，底池 = 19.5BB

**河牌后底池**：
- **转牌后底池 + 河牌下注 + 跟注**：19.5 + 10 + 10 = 39.5
- **例子**：转牌后底池 19.5BB，下注 10BB，跟注 10BB，底池 = 39.5BB

### 底池计算技巧

**技巧 1：相对底池计算**

**方法**：使用相对底池（以 BB 为单位）计算，更直观

**例子**：
- **底池**：10BB
- **下注**：5BB（1/2 Pot）
- **跟注成本**：5BB
- **所需胜率**：5 / (10 + 5) = 33%

**技巧 2：快速估算**

**方法**：使用常见比例快速估算

**例子**：
- **底池**：约 100
- **对手下注**：约 50（1/2 Pot）
- **所需胜率**：约 33%（记忆：1/2 Pot ≈ 33%）

**技巧 3：精确计算**

**方法**：在重要决策点使用精确计算

**例子**：
- **底池**：100
- **对手下注**：50
- **跟注成本**：50
- **所需胜率**：50 / (100 + 50 + 50) = 25%

## 底池计算在 GTO 决策中的应用（分）

### 应用 1：判断是否跟注

**步骤**：
1. **计算底池大小**：估算当前底池
2. **计算跟注成本**：计算需要投入的筹码
3. **计算所需胜率**：使用公式计算所需胜率
4. **估算实际胜率**：估算自己在对手范围下的胜率
5. **比较判断**：如果实际胜率 > 所需胜率，跟注；否则弃牌

**例子**：
- **底池**：100
- **对手下注**：50
- **跟注成本**：50
- **所需胜率**：25%
- **实际胜率**：估算约 35%
- **判断**：35% > 25%，应该跟注

### 应用 2：选择下注尺度

**步骤**：
1. **分析范围优势**：判断范围优势是否明显
2. **选择下注尺度**：根据范围优势选择尺度
3. **计算底池大小**：计算下注后的底池大小
4. **验证 EV**：使用 EV 验证下注尺度是否合理

**例子**：
- **范围优势**：明显
- **选择尺度**：1/2 Pot（50%）
- **底池**：100
- **下注**：50
- **下注后底池**：150
- **EV 验证**：下注 EV 为正，尺度合理

### 应用 3：计算 EV

**步骤**：
1. **计算底池大小**：估算当前底池
2. **计算下注成本**：计算需要投入的筹码
3. **估算胜率**：估算自己在对手范围下的胜率
4. **计算 EV**：使用 EV 公式计算
5. **判断决策**：如果 EV > 0，执行；否则不执行

**例子**：
- **底池**：100
- **下注**：50
- **胜率**：60%
- **EV**：0.6 × 150 - 0.4 × 50 = 90 - 20 = 70
- **判断**：EV > 0，应该下注
