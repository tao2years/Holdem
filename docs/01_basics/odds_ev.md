# 赔率与期望值（基础）

本章以"总-分-总"结构详细整理赔率与期望值，为 GTO 与策略决策提供数理基础。

## 本节目的（总）

- 深入理解赔率与期望值的概念与计算方法
- 掌握赔率与胜率的关系，建立决策框架
- 提供系统化的训练方法与实战应用

## 赔率（分）

### 底池赔率

**底池赔率（Pot Odds）**：当前底池与跟注成本的比值。

**计算公式**：
```
底池赔率 = 底池大小 / 跟注成本
```

**表示方法**：
- **比例表示**：2:1、3:1、4:1 等
- **百分比表示**：33%、25%、20% 等

**例子**：
- **底池**：100
- **对手下注**：50
- **跟注成本**：50
- **底池赔率**：100 / 50 = 2:1（或 33%）

### 隐含赔率

**隐含赔率（Implied Odds）**：考虑未来可能赢得的底池。

**计算公式**：
```
隐含赔率 = (当前底池 + 未来可能赢得的底池) / 跟注成本
```

**使用场景**：
- **听牌**：有听牌时，考虑未来可能赢得的底池
- **多街对局**：在多街对局中，考虑后续街的底池增长
- **对手类型**：面对松手时，隐含赔率更高

**例子**：
- **当前底池**：100
- **跟注成本**：50
- **未来可能赢得**：200（如果听牌成功）
- **隐含赔率**：(100 + 200) / 50 = 6:1

### 反向隐含赔率

**反向隐含赔率（Reverse Implied Odds）**：考虑未来可能损失的底池。

**计算公式**：
```
反向隐含赔率 = (当前底池 - 未来可能损失的底池) / 跟注成本
```

**使用场景**：
- **弱牌**：手牌较弱时，考虑未来可能损失
- **对手类型**：面对紧手时，反向隐含赔率更高
- **多街对局**：在多街对局中，考虑后续街的损失

**例子**：
- **当前底池**：100
- **跟注成本**：50
- **未来可能损失**：100（如果对手更强）
- **反向隐含赔率**：(100 - 100) / 50 = 0:1（不应该跟注）

## 期望值（EV）（分）

### EV 的定义

**期望值（Expected Value, EV）**：在不确定性下的平均收益。

**计算公式**：
```
EV = (胜率 × 收益) - (败率 × 损失)
```

**简化公式**：
```
EV = 胜率 × (底池 + 下注) - 败率 × 下注
```

**例子**：
- **底池**：100
- **下注**：50
- **胜率**：60%
- **EV**：0.6 × (100 + 50) - 0.4 × 50 = 90 - 20 = 70

### EV 的判断标准

**正 EV（+EV）**：
- **含义**：该选择在长期有利
- **决策**：应该执行该选择
- **例子**：EV = +70，应该下注

**负 EV（-EV）**：
- **含义**：该选择在长期不利
- **决策**：不应该执行该选择
- **例子**：EV = -20，不应该跟注

**零 EV（0 EV）**：
- **含义**：该选择在长期平衡
- **决策**：可以执行，但不影响长期收益
- **例子**：EV = 0，可以跟注也可以弃牌

### EV 的计算方法

**方法 1：直接计算**

**步骤**：
1. **确定胜率**：估算自己在对手范围下的胜率
2. **确定收益**：计算获胜时的收益
3. **确定损失**：计算失败时的损失
4. **计算 EV**：使用公式计算

**例子**：
- **胜率**：60%
- **收益**：150（底池 100 + 下注 50）
- **损失**：50（下注 50）
- **EV**：0.6 × 150 - 0.4 × 50 = 90 - 20 = 70

**方法 2：简化计算**

**步骤**：
1. **计算底池赔率**：底池大小 / 跟注成本
2. **计算所需胜率**：跟注成本 / (底池 + 跟注成本)
3. **比较实际胜率**：如果实际胜率 > 所需胜率，EV 为正

**例子**：
- **底池**：100
- **跟注成本**：50
- **所需胜率**：50 / (100 + 50) = 33%
- **实际胜率**：60%
- **判断**：60% > 33%，EV 为正

## 赔率与胜率关系（分）

### 跟注所需胜率

**跟注所需胜率（Required Equity）**：跟注所需的最低胜率。

**计算公式**：
```
所需胜率 = 跟注成本 / (底池 + 对手下注 + 跟注成本)
```

**简化公式**：
```
所需胜率 = 跟注成本 / (底池 + 跟注成本)
```

**例子**：
- **底池**：100
- **对手下注**：50
- **跟注成本**：50
- **所需胜率**：50 / (100 + 50 + 50) = 25%

### 胜率与赔率的关系

**核心原则**：当估算胜率高于所需胜率时，长期 EV 为正。

**判断标准**：
- **实际胜率 > 所需胜率**：EV 为正，应该跟注
- **实际胜率 = 所需胜率**：EV 为零，可以跟注
- **实际胜率 < 所需胜率**：EV 为负，不应该跟注

**例子**：
- **所需胜率**：25%
- **实际胜率**：35%
- **判断**：35% > 25%，EV 为正，应该跟注

### 常见下注尺度的所需胜率

**1/3 Pot（33%）下注**：
- **所需胜率**：约 25%
- **记忆**：1/3 Pot ≈ 25% 胜率

**1/2 Pot（50%）下注**：
- **所需胜率**：约 33%
- **记忆**：1/2 Pot ≈ 33% 胜率

**2/3 Pot（67%）下注**：
- **所需胜率**：约 40%
- **记忆**：2/3 Pot ≈ 40% 胜率

**1 Pot（100%）下注**：
- **所需胜率**：约 50%
- **记忆**：1 Pot ≈ 50% 胜率

## 常见场景（分）

### 场景 1：面对下注

**决策流程**：
1. **计算底池大小**：估算当前底池
2. **计算跟注成本**：计算需要投入的筹码
3. **计算所需胜率**：使用公式计算所需胜率
4. **估算实际胜率**：估算自己在对手范围下的胜率
5. **比较判断**：如果实际胜率 > 所需胜率，跟注；否则弃牌

**例子**：
- **底池**：100
- **对手下注**：50
- **跟注成本**：50
- **所需胜率**：25%
- **实际胜率**：估算约 35%（同花听牌）
- **判断**：35% > 25%，应该跟注

### 场景 2：下注决策

**决策流程**：
1. **分析范围优势**：判断范围优势是否明显
2. **选择下注尺度**：根据范围优势选择尺度
3. **估算对手跟注范围**：估算对手可能跟注的范围
4. **计算 EV**：计算下注的期望值
5. **判断决策**：如果 EV > 0，下注；否则过牌

**例子**：
- **范围优势**：明显
- **选择尺度**：1/2 Pot（50%）
- **对手跟注范围**：估算约 40%
- **下注 EV**：计算为正
- **判断**：EV > 0，应该下注

### 场景 3：加注决策

**决策流程**：
1. **分析手牌强度**：判断手牌是否足够强
2. **估算对手范围**：估算对手可能的范围
3. **计算加注 EV**：计算加注的期望值
4. **比较跟注 EV**：比较加注与跟注的 EV
5. **判断决策**：如果加注 EV > 跟注 EV，加注；否则跟注

**例子**：
- **手牌强度**：强（AA）
- **对手范围**：估算较宽
- **加注 EV**：计算为正且较高
- **跟注 EV**：计算为正但较低
- **判断**：加注 EV > 跟注 EV，应该加注
