# 范围表与训练工具

本章以"总-分-总"结构提供翻牌前范围表、训练清单与实用工具，帮助系统化掌握 GTO 范围。

## 本节目的（总）

- 提供标准化的翻牌前范围表（按位置与行动）
- 建立范围训练清单，便于日常练习
- 提供范围构建与验证工具

## 翻牌前范围表（分）

### 范围表示方法

- **100%**：该手牌组合 100% 执行该行动
- **50%**：该手牌组合 50% 执行该行动（混合策略）
- **空白**：该手牌组合不执行该行动

### 按钮位（BTN）范围

#### 未有人加注（Open Raise）

| 手牌类型 | 范围 | 说明 |
|---------|------|------|
| 对子 | 22+ | 所有对子 |
| 同花连张 | 54s+ | 54s 及以上同花连张 |
| 隔张同花 | 64s+ | 64s 及以上隔张同花 |
| 高牌 | A2s+, K2s+, Q2s+, J2s+, T2s+, 92s+, 82s+, 72s+ | 所有同花高牌 |
| 高牌非同花 | A2o+, K9o+, Q9o+, J9o+, T9o+ | 高牌非同花 |
| 连张非同花 | 54o+ | 54o 及以上连张 |

#### 面对加注（3-Bet）

- **价值 3-Bet**：QQ+, AK（100%）
- **混合 3-Bet**：JJ, AQ（50%），ATs, KQs（30%）
- **诈唬 3-Bet**：A5s, A4s（50%），K9s, Q9s（30%）

#### 面对 3-Bet（4-Bet / Fold）

- **4-Bet 价值**：AA, KK（100%），AK（80%），QQ（50%）
- **4-Bet 诈唬**：A5s（30%），A4s（20%）
- **跟注**：JJ, TT（100%），99（50%），AQ（80%）

### 小盲位（SB）范围

#### 面对按钮加注

- **3-Bet 价值**：QQ+, AK（100%）
- **3-Bet 混合**：JJ, AQ（50%），ATs（30%）
- **跟注**：22-99, A2s-A9s, K2s-K9s, Q2s-Q9s, J2s-J9s, T2s-T9s, 92s-82s, 72s（按位置调整频率）

### 大盲位（BB）范围

#### 面对加注（Call / 3-Bet / Fold）

- **3-Bet 价值**：QQ+, AK（100%）
- **3-Bet 混合**：JJ, AQ（50%），ATs（30%）
- **跟注**：22-99, A2s-A9s, K2s-K9s, Q2s-Q9s, J2s-J9s, T2s-T9s, 92s-82s, 72s, A2o-A9o, K2o-K9o, Q2o-Q9o, J2o-J9o, T2o-T9o（按位置调整频率）

### 中位（MP）范围

#### 未有人加注（Open Raise）

- **范围**：比按钮位更紧，去掉边缘手牌
- **对子**：22+（100%）
- **同花**：A2s+, K2s+, Q2s+, J2s+, T2s+, 92s+, 82s+, 72s+（部分混合）
- **高牌**：A2o+, K9o+, Q9o+, J9o+, T9o+（部分混合）

### 前位（EP）范围

#### 未有人加注（Open Raise）

- **范围**：最紧，只玩强牌
- **对子**：22+（100%）
- **同花**：A2s+, K2s+, Q2s+, J2s+, T2s+（部分混合）
- **高牌**：A2o+, K9o+, Q9o+（部分混合）

## 范围训练清单（分）

### 日常训练任务

#### 任务 1：位置范围记忆

- **目标**：熟记各位置的翻牌前范围
- **方法**：
  1. 每天复习一个位置的完整范围
  2. 随机抽取手牌，判断是否在范围内
  3. 记录错误，重点复习

#### 任务 2：3-Bet 范围构建

- **目标**：掌握面对加注时的 3-Bet 范围
- **方法**：
  1. 列出价值 3-Bet 手牌（100%）
  2. 列出混合 3-Bet 手牌（50%）
  3. 列出诈唬 3-Bet 手牌（30%）
  4. 验证范围平衡性（价值:诈唬 ≈ 2:1）

#### 任务 3：范围频率计算

- **目标**：快速计算范围频率
- **方法**：
  1. 给定范围，计算包含的手牌组合数
  2. 计算占所有可能手牌的比例
  3. 练习：按钮位 Open 范围频率是多少？

### 进阶训练任务

#### 任务 4：范围对比分析

- **目标**：对比不同 Solver 或来源的范围差异
- **方法**：
  1. 收集多个来源的范围表
  2. 对比差异，分析原因
  3. 理解不同场景下的范围调整

#### 任务 5：范围演化追踪

- **目标**：理解范围在多街中的演化
- **方法**：
  1. 选择翻牌前范围
  2. 给定翻牌，计算翻后范围
  3. 追踪范围在转牌、河牌的进一步演化

## 范围构建工具（分）

### 手牌组合计算

- **对子**：6 种组合（如 AA：AsAh, AsAd, AsAc, AhAd, AhAc, AdAc）
- **同花**：4 种组合（如 AKs：AsKs, AhKh, AdKd, AcKc）
- **非同花**：12 种组合（如 AKo：AsKh, AsKd, AsKc, AhKs, AhKd, AhKc, AdKs, AdKh, AdKc, AcKs, AcKh, AcKd）

### 范围频率计算

- **总组合数**：1326 种（52 选 2，不考虑顺序）
- **范围频率**：范围包含的组合数 / 1326

### 范围可视化工具

- **范围表工具**：使用在线工具或软件绘制范围表
- **组合计算器**：快速计算手牌组合数
- **频率计算器**：计算范围频率与胜率

## 训练建议（分）

### 记忆方法

1. **分类记忆**：按对子、同花、高牌分类记忆
2. **位置对比**：对比不同位置的范围差异
3. **频率理解**：理解混合策略的频率含义

### 验证方法

1. **随机测试**：随机抽取手牌，判断是否在范围内
2. **范围构建**：给定场景，构建完整范围
3. **频率计算**：计算范围的组合数与频率

### 进阶方向

1. **动态调整**：根据对手类型调整范围
2. **多街演化**：理解范围在多街中的变化
3. **平衡验证**：验证范围的平衡性（价值:诈唬比例）

## 总结（总）

范围表与训练工具是掌握 GTO 翻牌前策略的基础。通过系统化的范围表、训练清单与实用工具，可以：

- **建立标准**：掌握各位置的标准范围
- **系统训练**：通过训练清单逐步提升
- **工具辅助**：使用工具验证与优化范围

持续练习与验证，将范围知识转化为实战能力。
