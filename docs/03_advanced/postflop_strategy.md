# 翻后策略框架

本章以“总-分-总”结构梳理翻牌后到河牌的策略框架，强调范围、频率与尺度的联动。

## 本节目的

- 建立翻后决策的通用流程
- 给出各街的重点关注事项

## 通用流程

1. 确定双方范围与优势
2. 评估牌面结构（干燥/湿润）
3. 分配下注与过牌频率
4. 选择下注尺度并计算 EV

## 翻牌（Flop）

- 更多使用小尺度高频下注
- 保护范围，避免过度暴露

### 翻牌训练重点

- 区分范围优势与牌面优势
- 选择高频小尺度或低频大尺度模型

## 转牌（Turn）

- 继续下注更偏极化
- 强化价值与诈唬的边界

### 转牌训练重点

- 识别提高/削弱范围优势的牌
- 将中等强度牌归入过牌范围

## 河牌（River）

- 价值与诈唬比例更接近均衡
- 下注尺度与对手范围强相关

### 河牌训练重点

- 价值下注与诈唬下注的比例管理
- 以 EV 为准判断是否下注或过牌

## 总结

翻后策略的核心是“范围优势 + 牌面结构 + 频率分配”，并用 EV 校验尺度选择。
