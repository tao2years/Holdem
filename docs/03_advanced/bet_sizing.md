# 下注尺度

本章以"总-分-总"结构详细整理下注尺度的策略逻辑、常见模式与训练方法。

## 本节目的（总）

- 深入理解下注尺度的作用与选择原则
- 说明尺度与范围强度的关系
- 提供系统化的尺度训练方法

## 核心逻辑（分）

### 下注尺度的作用

**作用 1：控制底池大小**

**小尺度（33-50%）**：
- **目的**：保持底池可控
- **适用**：范围优势明显，需要保护范围
- **效果**：对手容易跟注，底池增长缓慢

**中等尺度（50-75%）**：
- **目的**：平衡价值与诈唬
- **适用**：范围优势适中，需要平衡
- **效果**：给对手不合适但可接受的赔率

**大尺度（75-100%）**：
- **目的**：最大化价值或迫使弃牌
- **适用**：范围极化，需要最大化价值
- **效果**：给对手极不合适赔率，迫使弃牌

**作用 2：给对手不合适赔率**

**赔率关系**：
- **小尺度**：给对手合适赔率，容易跟注
- **中等尺度**：给对手不合适但可接受的赔率
- **大尺度**：给对手极不合适赔率，迫使弃牌

**实际应用**：
- **价值下注**：使用中等或大尺度，给对手不合适赔率
- **诈唬下注**：使用大尺度，迫使对手弃牌
- **保护范围**：使用小尺度，保护中等牌力

**作用 3：平衡价值与诈唬**

**价值:诈唬比例**：
- **小尺度（33%）**：价值:诈唬 ≈ 3:1
- **中等尺度（50%）**：价值:诈唬 ≈ 2:1
- **大尺度（75%）**：价值:诈唬 ≈ 1:1
- **全下（100%）**：价值:诈唬 ≈ 1:1

**平衡原则**：
- **价值牌多**：使用小或中等尺度
- **价值牌少**：使用大尺度
- **范围极化**：使用大尺度

### 大尺度用于极强与极弱（诈唬）范围

**极强范围**：
- **特点**：手牌极强，几乎必胜
- **策略**：使用大尺度，最大化价值
- **例子**：AA, KK, 三条, 同花顺

**极弱范围（诈唬）**：
- **特点**：手牌极弱，需要迫使对手弃牌
- **策略**：使用大尺度，迫使对手弃牌
- **例子**：听牌失败，完全空气

**原因**：
- **极强范围**：对手跟注概率低，需要大尺度获取价值
- **极弱范围**：对手跟注概率高，需要大尺度迫使弃牌

### 小尺度用于薄价值与高频试探

**薄价值**：
- **特点**：手牌有一定强度，但不够强
- **策略**：使用小尺度，获取薄价值
- **例子**：顶对弱踢脚，中等对子

**高频试探**：
- **特点**：范围优势明显，需要高频下注
- **策略**：使用小尺度，配合高频下注
- **例子**：干燥牌面，范围优势明显

**原因**：
- **薄价值**：对手跟注概率高，小尺度可以获取更多价值
- **高频试探**：小尺度可以保护范围，避免过度暴露

## 常见尺度选择（分）

### 1/3 Pot（33%）

**使用场景**：
- **范围优势明显**：我的范围在牌面上明显更有优势
- **干燥牌面**：连接性弱、同花可能低
- **高频下注**：需要高频下注保护范围

**策略特点**：
- **下注频率**：70-80%
- **价值:诈唬比例**：约 3:1
- **目的**：保护范围，控制底池

**实际例子**：
- **翻牌 A♠7♦2♣，BTN vs BB**：
  - BTN 范围优势明显
  - 使用 33% 尺度，下注频率 80%
  - 保护范围，控制底池

### 1/2 Pot（50%）

**使用场景**：
- **范围优势适中**：我的范围在牌面上有一定优势
- **中等牌面**：连接性中等，听牌适中
- **平衡价值与诈唬**：需要平衡价值与诈唬

**策略特点**：
- **下注频率**：50-60%
- **价值:诈唬比例**：约 2:1
- **目的**：平衡价值与诈唬，获取价值

**实际例子**：
- **翻牌 K♠9♦4♣，BTN vs BB**：
  - BTN 范围优势适中
  - 使用 50% 尺度，下注频率 60%
  - 平衡价值与诈唬

### 2/3 Pot（67%）

**使用场景**：
- **范围优势不明显**：双方范围在牌面上接近
- **湿润牌面**：连接性强、听牌多
- **偏极化下注**：范围极化，需要大尺度

**策略特点**：
- **下注频率**：40-50%
- **价值:诈唬比例**：约 1:1
- **目的**：给听牌不合适赔率，最大化价值

**实际例子**：
- **翻牌 9♠8♦7♣，BTN vs BB**：
  - BTN 范围优势不明显
  - 使用 67% 尺度，下注频率 50%
  - 给听牌不合适赔率

### 1 Pot（100%）

**使用场景**：
- **范围极度极化**：只有强牌和弱牌
- **河牌价值下注**：最大化价值获取
- **强诈唬**：需要迫使对手弃牌

**策略特点**：
- **下注频率**：30-40%
- **价值:诈唬比例**：约 1:1
- **目的**：最大化价值或迫使弃牌

**实际例子**：
- **河牌 A♠K♠Q♠J♠10♠，BTN vs BB**：
  - BTN 有同花顺或完全空气
  - 使用 100% 尺度，下注频率 30%
  - 最大化价值或迫使弃牌

## 尺度与范围强度的关系（分）

### 范围强度分类

**极强范围**：
- **特点**：手牌极强，几乎必胜
- **策略**：使用大尺度（75-100%），最大化价值
- **例子**：AA, KK, 三条, 同花顺

**强范围**：
- **特点**：手牌强，有很大优势
- **策略**：使用中等尺度（50-75%），获取价值
- **例子**：顶对强踢脚, 两对, 三条

**中等范围**：
- **特点**：手牌有一定强度，但不够强
- **策略**：使用小尺度（33-50%），获取薄价值
- **例子**：顶对弱踢脚, 中等对子

**弱范围（诈唬）**：
- **特点**：手牌弱，需要迫使对手弃牌
- **策略**：使用大尺度（75-100%），迫使弃牌
- **例子**：听牌失败, 完全空气

### 尺度选择原则

**原则 1：根据范围强度选择尺度**

**极强范围**：
- **尺度**：75-100%
- **原因**：对手跟注概率低，需要大尺度获取价值

**强范围**：
- **尺度**：50-75%
- **原因**：对手跟注概率适中，中等尺度可以获取价值

**中等范围**：
- **尺度**：33-50%
- **原因**：对手跟注概率高，小尺度可以获取更多价值

**弱范围（诈唬）**：
- **尺度**：75-100%
- **原因**：对手跟注概率高，需要大尺度迫使弃牌

**原则 2：根据范围优势选择尺度**

**范围优势明显**：
- **尺度**：33-50%（小尺度）
- **原因**：可以高频下注，小尺度保护范围

**范围优势适中**：
- **尺度**：50-75%（中等尺度）
- **原因**：需要平衡价值与诈唬

**范围优势不明显**：
- **尺度**：50-75%（中等尺度）
- **原因**：需要给听牌不合适赔率

**范围极化**：
- **尺度**：75-100%（大尺度）
- **原因**：只有强牌和弱牌，需要大尺度

## 训练方法（分）

### 方法 1：每个街道只练两种尺度

**训练策略**：
- **翻牌**：练习 33% 和 50% 两种尺度
- **转牌**：练习 50% 和 75% 两种尺度
- **河牌**：练习 50% 和 100% 两种尺度

**训练步骤**：
1. **选择场景**：选择一个具体的场景
2. **确定范围**：确定我的范围和对手范围
3. **选择尺度**：根据范围优势选择尺度
4. **验证 EV**：使用 EV 验证尺度是否合理

**训练目标**：
- **快速判断**：能够在 10 秒内判断合适的尺度
- **准确选择**：能够准确选择尺度，准确率 > 80%

### 方法 2：同一牌面保持尺度一致

**一致性原则**：
- **同一牌面**：在同一个牌面上，保持尺度一致
- **不同手牌**：不同手牌可以使用相同尺度
- **避免暴露**：避免通过尺度暴露手牌强度

**实际应用**：
- **翻牌 A♠7♦2♣**：所有下注都使用 33% 尺度
- **转牌 K♥**：所有下注都使用 50% 尺度
- **河牌 Q♣**：所有下注都使用 50% 尺度

**训练方法**：
- **选择牌面**：选择一个牌面
- **确定尺度**：根据范围优势确定尺度
- **保持一致**：在该牌面上保持尺度一致
- **验证效果**：验证一致性是否有效

### 方法 3：结合对手倾向调整尺度

**对手类型**：
- **紧手**：面对下注容易弃牌
- **松手**：面对下注容易跟注
- **激进**：面对下注容易加注

**尺度调整**：
- **面对紧手**：使用小尺度，获取更多价值
- **面对松手**：使用大尺度，迫使弃牌或获取更多价值
- **面对激进**：使用中等尺度，避免被加注

**训练方法**：
- **识别对手类型**：快速识别对手类型
- **调整尺度**：根据对手类型调整尺度
- **验证效果**：验证调整是否有效

## 实战提示（分）

### 提示 1：结合对手倾向调整尺度

**紧手**：
- **特点**：面对下注容易弃牌
- **策略**：使用小尺度，获取更多价值
- **例子**：33% 尺度，获取薄价值

**松手**：
- **特点**：面对下注容易跟注
- **策略**：使用大尺度，获取更多价值或迫使弃牌
- **例子**：75% 尺度，最大化价值

**激进**：
- **特点**：面对下注容易加注
- **策略**：使用中等尺度，避免被加注
- **例子**：50% 尺度，平衡价值与诈唬

### 提示 2：与范围频率协同考虑

**频率与尺度的关系**：
- **高频小尺度**：范围优势明显，高频下注，小尺度保护范围
- **中频中等尺度**：范围优势适中，中频下注，中等尺度平衡
- **低频大尺度**：范围极化，低频下注，大尺度最大化价值

**协同原则**：
- **先确定频率**：根据范围优势确定下注频率
- **再选择尺度**：根据频率和范围强度选择尺度
- **最后验证**：使用 EV 验证频率和尺度是否合理

### 提示 3：使用 EV 验证尺度

**EV 验证步骤**：
1. **计算下注 EV**：计算下注的期望值
2. **计算过牌 EV**：计算过牌的期望值
3. **比较选择**：选择 EV 更高的行动
4. **调整尺度**：如果 EV 为负，调整尺度

**实际应用**：
- **下注 EV > 过牌 EV**：应该下注，尺度合理
- **下注 EV = 过牌 EV**：可以下注也可以过牌
- **下注 EV < 过牌 EV**：应该过牌，或调整尺度

- **尺度选择**：根据范围强度、范围优势、对手类型选择尺度
- **训练方法**：每个街道只练两种尺度，同一牌面保持尺度一致
- **实战提示**：结合对手倾向调整尺度，与范围频率协同考虑，使用 EV 验证
