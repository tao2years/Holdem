# 进阶模块总览

本章以“总-分-总”方式整理进阶模块入口，并说明后续扩展方向。

## 本节目的

- 明确进阶模块范围
- 给出阅读顺序

## 阅读顺序

1. `strategy_models.md`：策略模型
2. `preflop_ranges.md`：翻牌前范围（位置与行动）
3. `postflop_strategy.md`：翻后策略框架
4. `bet_sizing.md`：下注尺度
5. `hand_reading_pot_dynamics.md`：底池动态与局势判断
6. `case_studies.md`：实战案例与训练任务
7. `training_checklists.md`：GTO 训练清单

## 总结

进阶模块将聚焦范围构建、下注尺度与对抗策略，为 Solver 抽象与验证提供理论支持。
