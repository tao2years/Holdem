# 实验记录模板

本模板以"总-分-总"结构提供标准化的实验记录格式，确保实验可复现、可对比。

## 实验信息（总）

### 基本信息

- **实验编号**：EXP-XXX
- **实验名称**：简短描述实验目标
- **创建日期**：YYYY-MM-DD
- **实验人员**：姓名
- **状态**：进行中 / 已完成 / 已取消

### 实验目标

- **主要目标**：明确实验要解决的核心问题
- **次要目标**：实验的附加目标
- **假设**：实验前的假设或预期

## 实验配置（分）

### 目标

详细描述实验要验证的问题或要达成的目标。

### 模型

- **算法**：使用的算法（如 CFR、CFR+、MCCFR）
- **实现**：代码文件路径
- **版本**：代码版本或 Git 提交哈希

### 参数

| 参数名称 | 参数值 | 说明 |
|---------|--------|------|
| 训练轮次 | 50000 | 迭代次数 |
| 随机种子 | 42 | 确保可复现 |
| 学习率 | 默认 | 算法学习率 |
| 硬件 | CPU 单线程 | 训练环境 |

### 数据

- **输入数据**：游戏定义、初始状态等
- **输出数据**：策略文件、日志文件、报告文件
- **数据路径**：数据存储路径

### 环境

- **Python 版本**：3.8+
- **依赖包**：列出关键依赖及版本
- **硬件配置**：CPU、内存、GPU（如适用）
- **操作系统**：Windows / Linux / macOS

## 过程记录（分）

### 训练轮次

- **总轮次**：实际训练的轮次
- **检查点**：检查点设置（如每 1000 轮）
- **训练时间**：总训练时间
- **平均每轮时间**：单轮平均时间

### 收敛表现

- **前 N 轮**：策略快速变化，后悔值大幅下降
- **N-M 轮**：策略逐步稳定，频率波动减小
- **M+ 轮**：策略基本稳定，变化 < X%

### 关键观察

1. **策略收敛速度**：约 N 轮后策略基本稳定
2. **后悔值趋势**：后悔值在前 N 轮快速下降，后续缓慢收敛
3. **频率分布**：各信息集合的策略频率逐步接近理论值
4. **其他观察**：任何值得注意的现象

### 异常情况

- **错误信息**：如有错误，记录错误信息
- **异常行为**：如有异常行为，详细描述
- **处理方式**：如何处理异常

## 结果与结论（分）

### 指标

#### 策略准确性

- **与理论解对比**：策略与理论最优策略的差异
- **关键信息集合**：列出关键信息集合的策略频率
- **差异分析**：分析差异原因

#### 收敛性指标

- **策略稳定性**：最后 N 轮策略变化 < X%
- **与理论解差异**：平均差异 < X%
- **期望收益**：玩家期望收益（零和博弈应接近 0）

#### 性能指标

- **训练时间**：总训练时间
- **内存占用**：峰值内存占用
- **CPU 使用率**：平均 CPU 使用率

### 可视化

- **策略树**：策略树可视化（如有）
- **收敛曲线**：收敛曲线图（如有）
- **频率分布**：策略频率分布图（如有）

### 结论

1. **主要结论**：实验的主要发现
2. **次要结论**：实验的附加发现
3. **问题发现**：发现的问题或不足

### 下一步

1. **短期计划**：下一步要做的实验或改进
2. **长期计划**：长期的研究方向
3. **待验证问题**：需要进一步验证的问题

## 数据文件（分）

### 策略文件

- **路径**：`data/strategies/experiment_name/strategy_XXX.json`
- **格式**：JSON，包含各信息集合的策略分布
- **大小**：文件大小

### 日志文件

- **路径**：`data/logs/experiment_name/training.log`
- **内容**：训练过程日志、策略变化记录
- **大小**：文件大小

### 报告文件

- **路径**：`data/reports/experiment_name/report.md`
- **内容**：实验报告
- **格式**：Markdown

### 可视化文件

- **策略树**：`data/reports/experiment_name/strategy_tree.png`
- **收敛曲线**：`data/reports/experiment_name/convergence.png`
- **频率分布**：`data/reports/experiment_name/frequency_dist.png`

## 对比分析（分）

### 与基准对比

- **基准实验**：对比的基准实验编号
- **差异分析**：与基准的差异及原因
- **改进效果**：改进的效果（如有）

### 与其他算法对比

- **对比算法**：对比的其他算法
- **性能对比**：性能对比结果
- **策略对比**：策略对比结果

## 总结（总）

### 实验总结

简要总结实验的主要发现、结论与下一步计划。

### 经验教训

- **成功经验**：实验中的成功经验
- **失败教训**：实验中的失败教训
- **改进建议**：对后续实验的改进建议

### 参考资料

- **相关论文**：相关的学术论文
- **相关实验**：相关的其他实验
- **相关文档**：相关的文档链接

## 使用说明

1. **复制模板**：复制本模板创建新实验记录
2. **填写信息**：按照模板填写实验信息
3. **定期更新**：实验过程中定期更新记录
4. **归档管理**：实验完成后归档到对应目录

## 注意事项

- **及时记录**：实验过程中及时记录，避免遗忘
- **详细描述**：关键步骤和观察要详细描述
- **数据备份**：重要数据要及时备份
- **版本控制**：使用 Git 管理实验代码和数据
